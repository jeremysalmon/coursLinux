Cours Linux 101 (Sommaire)

Gestion des utilisateurs

sudo : devenir super utilisateur

su : devenir super utilisateur et le rester

adduser : ajouter un utilisateur
ex : adduser mohamed

passwd : changer le mot de passe d'un utilisateur
ex : passwd mohamed

deluser : supprimer un compte utilisateur
ex : deluser mohamed
ex : deluser --remove-home mohamed

addgroup : ajouter un groupe
ex : addgroup monGroupe

usermod : modifie un utilisateur
ex : usermod -l mohamed reda <- renomme l'utilisateur mohamed en reda
ex : usermod -g monGroupe mohamed <- change le groupe de mohamed
ex : usermod -G monGroupe,www-data mohamed <- change et associe plusieurs groupes à mohamed
ex : usermod -aG postfix mohamed <- ajoute un groupe à mohamed

delgroup : supprime un groupe

chown : changer le propriétaire d'un fichier / répertoires
-R pour le récursif

chgroup : changer le groupe d'un fichier
chown peut également changer un groupe au format : chown groupe:utilisateur fichier

chmod : chnager les droits sur un fichier

Un droit de lecture à la valeur 4 (r)
Un droit d'écriture à la valeur 2 (w)
Un droit d'execution à la valeur 1 (x)
Si nous voulons donner les droits en lecture/ecriture/execution c'est donc un 7.
Si nous voulons donner les droits en lecture c'est un 4

Un fichier à 3 types de droits :

  • Pour le propriétaire
  • Pour le groupe
  • Pour les autres

Cours Linux 101 (Sommaire)

La première étape est de connaitre l'adresse IP de votre machine virtuelle.

Pour celà, dans la console de VirtualBox, connectez vous en root et tapez ifconfig

Vous trouverez l'adresse de votre Linux sur le réseau local

Lancez putty et connectez vous à l'adresse IP de votre machine Linux

Indiquer le nom de compte et le mot de passe

ATTENTION : Par défaut, vous ne pouvez pas vous connectez en root depuis Putty (en SSH)

Pour pouvoir vous connecter en root il faut :

  • Passer en administrateur : su
  • Editer le fichier de config de ssh : nano /etc/ssh/sshd_config
  • Changer la valeur de PermitRootLogin à Yes
  • Redémarrer le service SSH : service ssh restart

Vous pouvez maintenant utiliser votre compte root depuis SSH

Vous ête maintenant connecté à Linux depuis Windows.

Sur le panneau de gauche vos fichiers Windows, sur le panneau de droite les fichiers de la machine Linux.
